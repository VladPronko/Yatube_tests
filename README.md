## Ранее, в учебных целях, мною была создана социальная сеть для блоггеров - "Yatube" (Python 3.7 Django 2.2.19).
## В данном репозитории была реализована задача покрытия проекта "Yatube" следующими тестами:
- моделей приложения posts в Yatube (правильность отображения постов, названий групп, отображение значения поля __str__ в объектах моделей);
- доступности страниц и названия шаблонов приложения Posts проекта Yatube (с учетом прав доступа);
- правильности html-шаблонов, используемых во view-функциях;
- соответствует ли ожиданиям словарь context, передаваемый в шаблон при вызове;
- если при создании поста указать группу, то этот пост появляется на главной странице сайта, на странице выбранной группы и в профайле пользователя. А также проверка, что пост не попал в группу, для которой не был предназначен;
- при отправке валидной формы со страницы создания поста создаётся новая запись в базе данных, а при отправке валидной формы со страницы редактирования поста происходит изменение поста с post_id в базе данных.
